This plugin allows you to use WooCommerce coupons as giftcards that your customers can purchase and use on your WooCommerce webstore.

== Description ==
Contrast WooCommerce giftcards plugin is very lite plugin. It creates taxonomy term called "Giftcards" so you can tag your WooCommerce product as a giftcard. The plugin will also create a metabox for products, so you can tag it as a percent product or a fixed product. So basically customers who buys giftcards buys regular WooCommerce products. After the purchase, the plugin will create coupons and give them unique coupon names (codes). The plugin will also add order ID(s) to the just created coupon(s) description(s).
Customers can see their ordered coupons (giftcards) in my-account page or in order completed email.

If you have an interest in translating this plugin please let us know.
We do not offer any support for Contrast WooCommerce giftcards plugin.
This plugin has been tested with WordPress core version 4.5.x and WooCommerce 2.6.x so we don't promise backwards compatibility.

== Installation ==
1. Install the Contrast WooCommerce plugin
2. Activate the plugin
3. Create WooCommerce products and tag them as Giftcards
4. Use metafield to set the giftcard to be either percent product or fixed product
